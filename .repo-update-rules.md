**CRITICAL RULE (explicitly requested by owner MrSilhouette):**

For ANY future updates or additions to this repository:

- **NEVER delete, remove, overwrite, or alter any existing projects or subdirectories.**
  - Example: When adding a brand new project, do **not** touch, rename, or delete the existing `physics-calculator/` folder or any of its contents (index.html, its README, etc.).

- **Only add new projects** as *additional* new subdirectories at the root (e.g., `new-project-name/` containing its own index.html + README.md).

- **Only update the root README.md by appending**:
  - Add a new bullet/entry under the "## Current Projects" section.
  - Include the approximate addition date, a brief description of the new project and how it was created with Grok, and a link to the new subdirectory.
  - **Do not rewrite, replace, shorten, or remove any existing project descriptions or entries.** The list must grow by addition only, preserving the full history of all work done together.

- Always consult and follow the detailed policies in:
  - [SECURITY.md](SECURITY.md) (read-only archive, no forks, private recommended, share originals only, risks of modified HTML/JS).
  - The "## Repository Policy & Security (Important)" and "## How to Add Your Own Grok-Assisted Project" sections in the root [README.md](README.md).

This rule ensures the complete history and all prior collaborative projects (e.g. the deep math training work and PhysicsCalc) are permanently preserved.
